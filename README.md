# UnwatchHiddenQuests
Automatically untracks hidden tracked quests which on login.  There is a known bug where tracked quests can remain hidden even after completing or abandoning the quest, resulting in a severe performance loss if it builds up.

**UPDATED FOR MIDNIGHT 12.0**.

> [!IMPORTANT]
> # Setup / Installation
> - Click the green Code button > Download ZIP.
> - Extract the "UnwatchHiddenQuests" folder to the Interface > Addons folder in your _retail wow client.
> - Login to the game and check the chatbox when logging in.
